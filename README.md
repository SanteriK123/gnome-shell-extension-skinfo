# gnome-shell-extension-skinfo

Simple GNOME extension that uses API of [skinfo](skinfo.dy.fi) to create an indicator in the top panel with separate submenus for each restaurant.

# Installation

Simply run the command below:  
```
git clone git@github.com:SanteriK123/gnome-shell-extension-skinfo.git ~/.local/share/gnome-shell/extensions/skinfo@santerik123.github.com/
```
